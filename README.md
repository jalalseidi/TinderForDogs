# TinDog 🐾

A fun, responsive landing page for **TinDog** — a fictional Tinder-style app that helps dogs meet other dogs nearby.

## Overview

TinDog is a single-page marketing website built with **HTML5** and **Bootstrap 5**. It showcases a mobile app concept with sections for features, testimonials, pricing plans, and a footer.

## Sections

- **Hero / Title** — Headline with app store download buttons and a phone mockup image.
- **Features** — Three key selling points: ease of use, elite clientele, and a satisfaction guarantee.
- **Testimonial** — A customer quote with a profile image and press logo bar (TechCrunch, Mashable, Business Insider, TNW).
- **Pricing** — Three tiered plans:
  | Plan | Price | Highlights |
  |-----------|-------|--------------------------------|
  | Chihuahua | Free | 5 matches/day, 10 messages/day |
  | Labrador | $15/mo | Unlimited matches & messages |
  | Mastiff | $29/mo | Priority listing + unlimited |
- **Footer** — Copyright notice and navigation link columns.

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in any modern web browser — no build step required.

```bash
git clone https://github.com/your-username/tindog.git
cd tindog
open index.html
```

## Customisation

- **Colors / Gradient** — Edit the `.gradient-background` class in `css/style.css`.
- **Profile image size** — Adjust the `.profile-img` class in `css/style.css`.
- **Pricing** — Update plan names, prices, and features directly in the `#pricing` section of `index.html`.
